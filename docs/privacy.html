Privacy Policy

Last updated: March 2026

Phone Whisper is an Android dictation app that records speech, transcribes it, and inserts the result into text fields across apps. Your privacy is a core feature of the app, not an afterthought.

Data handling

Phone Whisper supports two transcription modes, giving you control over where your audio is processed.

Local mode

In local mode, audio is processed on-device using local speech recognition models. Audio does not leave your device. This is the recommended mode for maximum privacy.

Cloud mode

In cloud mode, recorded audio is sent directly from the device to OpenAI's transcription API to generate text.

If optional cleanup is enabled, the transcribed text is also sent directly from the device to OpenAI's chat API to improve punctuation, capitalization, and clarity.

API keys

If you use cloud features, your OpenAI API key is stored locally on your device in secure app storage. It is used only to authenticate requests sent directly from your device to OpenAI.

I do not operate a relay server. Your keys and your data never touch any infrastructure controlled by me.

Accessibility Service

Phone Whisper uses Android Accessibility Service for the sole purpose of identifying the currently focused text field and inserting dictated text. This happens only after you explicitly interact with the floating overlay button.

  • It does not monitor your browsing.
  • It does not collect screen content for analytics.
  • It does not perform background automation or "keylogging."

Data collection

I do not run a backend for Phone Whisper. I do not collect user accounts, analytics, crash reports, or uploaded recordings.

Third-party services you choose to use, such as OpenAI, may process data according to their own terms and privacy policies.

Open source

Because trust is earned, Phone Whisper is open source. You can audit the code yourself on GitHub to verify how your data and permissions are handled.
